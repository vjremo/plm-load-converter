Validation errors

    Output preview (tab-delimited)

    Column suffix rules

    -SingleList -ColorChoice

    Display name → internal name via References sheet. Error if not found.

    -MultiList -Composite

    Comma-separated display names → internal names joined by |~*~|.

    -Float

    Must be a valid decimal number. Error otherwise.

    -Boolean

    Yes → true, No → false. Only these two values accepted.

    -Integer

    Must be a whole number with no decimal point. Error otherwise.

    -MultiEntry

    Comma-separated plain text values joined by |~*~|. No reference lookup.
